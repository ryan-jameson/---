# 🍅 聚力专注 - 学习专注番茄钟小程序

「聚力专注」是一款基于**微信小程序原生语法 + 微信云开发**打造的自律与时间管理应用。旨在通过番茄工作法、目标拆解以及社交化的小组监督机制，帮助用户提高学习与工作效率。

## ✨ 核心功能清单

1. **🔐 用户认证体系**：基于微信云开发的静默授权登录机制。
2. **⏱️ 专业番茄钟**：标准 25分钟专注 + 5分钟休息，支持绑定目标、填写备注、随时暂停及结束。
3. **🎯 学习目标管理**：区分「终极目标」和「阶段性目标」，完善的目标状态跟进与拆解系统。
4. **👥 学习小组监督**：通过邀请码加入圈子，实时查看成员今日/本周的专注排行榜、目标达成率，并支持“点赞”与“鼓励留言墙”。
5. **📈 数据报表与总结**：自动按日、周、月维度聚合计算专注总时间与番茄次数，支持填写每日学习心得。

## 🚀 部署与运行流程

### 步骤1：导入项目
1. 下载代码，打开 **微信开发者工具**。
2. 点击「导入项目」，选择包含本代码的 `聚力` 文件夹。
3. 填入你在此前申请的**小程序 AppID**（需要有云开发权限），并在右下角后端服务选择「微信云开发」。

### 步骤2：配置云开发环境
1. 点击开发者工具顶部的 **「云开发」** 按钮，开通云开发服务。
2. 获取你的 **云环境 ID (Env ID)**。
3. 打开客户端项目中的 `app.js`，找到 `wx.cloud.init({ ... })`，将 `env` 字段的值填写为你自己的云环境 ID。

### 步骤3：初始化云函数依赖并拉取数据表结构
1. 如果之前只看到了【上传代码】的按钮而没有依赖选项，请先在开发者工具界面上方找到并打开 **“云开发控制台”**。
2. 确保在根目录下的 `project.config.json` 中配置了 `"cloudfunctionRoot": "cloudfunctions/"`，以便开发工具识别云函数目录。（该项如果已存在请忽略）
3. **关键：选择云环境**。在开发者工具左侧，找到带有小云朵图标的 `cloudfunctions` 根文件夹，**右键点击它** -> 选择 **「当前环境」** -> 勾选你开通的云环境（通常名字类似 `cloud1-xxx`）。选中后，该文件夹名字旁边会多出一个括号显示当前环境。
4. 我们已经在各个云函数目录为您补充了必需的 `package.json` 文件。现在展开 `cloudfunctions` 目录，右键点击里面的 `initDbCollections` 文件夹，选择 **「上传并部署：云端安装依赖 (不上传node_modules)」**。
5. 部署成功后，进入你的 **「云开发控制台」 -> 「云函数」** 列表，找到 `initDbCollections`，点击 **「云端测试」** 进行执行。
5. 看到执行成功后，工具已经自动为你的云数据库建立了 7 个基础集合结构：`users`, `targets`, `pomodoro`, `groups`, `group_members`, `daily_records`, `group_messages`。

> **⚠️核心数据结构权限配置**：为保证互助监控功能与获取自身记录，请在 “云数据库控制台” 的集合列表中，将所有刚刚建立的数据表，在「数据权限」处统一修改为：**「所有用户可读，仅创建者可读写」**。

### 步骤4：部署业务云函数并登录授权
依次右键点击下面每个业务云函数文件夹，选择 **「上传并部署：云端安装依赖」**（不要仅选择“上传代码”，务必有“安装依赖”这一项）：
* `cloudfunctions/login` （此云函数负责后台获取使用者的身份凭证 openid，若提示“请先授权”，多因未成功部署此项导致静默拿不到身份）
* `cloudfunctions/groupService` （此云函数为小组成员和互助社交的核心枢纽）

部署完成后，在开发者工具模拟器内按 `Ctrl+R`（或者点击栏目的编译按钮）刷新小程序进程，即可正常通过首屏点击授权获取身份。

## 🛠️ 调试流程与建议

1. **界面调试**：通过微信开发者工具的页面路径选择器，你可以直接跳转到 `/pages/index/index` 或 `/pages/group/group` 查看UI。如果图标显示为空，请从 iconfont 下载图标放入 `assets/icons` 中。
2. **深浅色模式调试**：点击开发者工具上方模拟器状态栏的 `☀️ / 🌙` 图标可直接测试暗黑模式下的组件显示效果。
3. **数据调试**：可以在云数据库里面手动修改 `pomodoro` 集合的专注时间，然后再回到小程序的 `pages/record` 刷新，测试周/月报表和榜单的金额计算是否正确。
4. **清除缓存**：如遇登录状态或授权无法弹起，可点击开发者工具顶层菜单【清缓存】 -> 【清除数据缓存 / 授权数据】进行重置再重试。
