# 🍅 聚力专注 - 学习专注番茄钟小程序

「聚力专注」是一款基于**微信小程序原生语法 + 微信云开发**打造的自律与时间管理应用。旨在通过番茄工作法、目标拆解以及社交化的小组监督机制，帮助用户提高学习与工作效率。

## ✨ 核心功能清单

1. **🔐 用户认证体系**：基于微信云开发的静默授权登录机制。
2. **⏱️ 专业番茄钟**：标准 25分钟专注 + 5分钟休息，支持绑定目标、填写备注、随时暂停及结束。
3. **🎯 学习目标管理**：区分「终极目标」和「阶段性目标」，完善的目标状态跟进与拆解系统。
4. **👥 学习小组监督**：通过邀请码加入圈子，实时查看成员今日/本周的专注排行榜、目标达成率，并支持“点赞”与“鼓励留言墙”。
5. **📈 数据报表与总结**：自动按日、周、月维度聚合计算专注总时间与番茄次数，支持填写每日学习心得。

## 🚀 部署与运行流程

### 步骤1：导入项目
1. 下载代码，打开 **微信开发者工具**。
2. 点击「导入项目」，选择包含本代码的 `聚力` 文件夹。
3. 填入你在此前申请的**小程序 AppID**（需要有云开发权限），并在右下角后端服务选择「微信云开发」。

### 步骤2：配置云开发环境
1. 点击开发者工具顶部的 **「云开发」** 按钮，开通云开发服务。
2. 获取你的 **云环境 ID (Env ID)**。
3. 打开客户端项目中的 `app.js`，找到 `wx.cloud.init({ ... })`，将 `env` 字段的值填写为你自己的云环境 ID。

### 步骤3：在云端初始化数据库
1. 在微信开发者工具的目录树中，右键点击 `cloudfunctions/initDbCollections` 文件夹，选择 **「上传并部署：云端安装依赖」**。
2. 部署完成后，在「云开发控制台」 -> 「云函数」中，找到 `initDbCollections`，点击 **「云端测试」**。
3. 运行后，程序将自动帮你在云数据库中建立以下 7 个集合结构：`users`, `targets`, `pomodoro`, `groups`, `group_members`, `daily_records`, `group_messages`。

> **⚠️权限设置**：为保证互助监控功能的运行，请在云数据库控制台中，将除了 `users` 表以外的其他相关数据表的「数据权限」修改为：**「所有用户可读，仅创建者可读写」**。

### 步骤4：部署业务云函数
依次右键点击以下云函数文件夹，选择 **「上传并部署：云端安装依赖」**（务必确保安装依赖，因引用了 `wx-server-sdk`）：
* `cloudfunctions/login`
* `cloudfunctions/groupService` （注意：此云函数为小组排行和交互的核心调度者）

## 🛠️ 调试流程与建议

1. **界面调试**：通过微信开发者工具的页面路径选择器，你可以直接跳转到 `/pages/index/index` 或 `/pages/group/group` 查看UI。如果图标显示为空，请从 iconfont 下载图标放入 `assets/icons` 中。
2. **深浅色模式调试**：点击开发者工具上方模拟器状态栏的 `☀️ / 🌙` 图标可直接测试暗黑模式下的组件显示效果。
3. **数据调试**：可以在云数据库里面手动修改 `pomodoro` 集合的专注时间，然后再回到小程序的 `pages/record` 刷新，测试周/月报表和榜单的金额计算是否正确。
4. **清除缓存**：如遇登录状态或授权无法弹起，可点击开发者工具顶层菜单【清缓存】 -> 【清除数据缓存 / 授权数据】进行重置再重试。
